# education-ml

## Project ##
This project covers a full Exploratory Data Analysis and data preparation pipeline, followed by the application of supervised classification machine learning algorithms to predict college student enrollment status. Three different classification models are trained and compared to determine which one performs best. This was completed as an assignment for the Machine Learning module, part of the Higher Diploma in Data Analytics at CCT College.

## Dataset ##
This dataset covers students across various undergraduate programs. It combines enrollment information including academic background, demographics, and socioeconomic factors with academic performance data.
Available at: https://archive.ics.uci.edu/dataset/697/predict+students+dropout+and+academic+success

## Algorithms ##
1. k-Nearest Neighbours
2. Decision Tree
3. Random Forest

## Hyperparameter Tuning ##
GridSearchCV
